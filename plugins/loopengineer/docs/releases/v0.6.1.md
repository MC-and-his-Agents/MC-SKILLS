# v0.6.1 Automatic Release Plan

Issue: #101

`v0.6.1` is the M8 release checkpoint for LoopEngineer automatic release
support. It records a fail-closed GitHub Actions release path that creates a
public tag and draft GitHub Release after release materials land on `main`.

## Included Scope

- Automatic release policy and manual fallback documentation.
- Automatic release planning mode for `scripts/prepare_manual_release.py`.
- `.github/workflows/auto-release.yml` for `main` release-material changes.
- README marketing badges and Project Records cleanup.
- Version, metadata, plugin manifest, changelog, and release evidence update to
  `0.6.1`.

## Not Included

- Package publication.
- Public release publication without maintainer review.
- Scheduler or watcher runtime.
- MCP mutation tools.
- Merge or closeout automation.
- Loom `.loom` writes or Loom adapter install/register.

## Compatibility

- pluginApiVersion: 1
- protocolVersion: 1
- engineContractVersion: 1
- schemaMajorVersion: 1
- skillContractVersion: 1
- adapterContractVersion: 0

`engineContractVersion: 1` remains compatible because M8 adds release
automation around repository facts and does not change the runtime-neutral
engine command envelope. `adapterContractVersion: 0` still means there is no
stable external runtime adapter contract.

## Release Evidence

Pre-merge validation must pass on the #101 closeout PR:

- `python3 -m unittest discover -s tests`
- `python3 scripts/validate_structures.py`
- `python3 scripts/check_release_readiness.py`
- `git diff --check`

Post-merge automatic release evidence must be recorded on #101 before closing
the M8 parent:

- target head SHA;
- automatic workflow run URL;
- `v0.6.1` tag URL;
- draft GitHub Release URL;
- release evidence artifact locator;
- remaining risk.

## Automatic Release Rule

After this release plan lands on `main`, `.github/workflows/auto-release.yml`
derives `v0.6.1` from `VERSION`, runs the fail-closed release plan, creates the
Git tag, and creates a draft GitHub Release. Package publication remains out of
scope.
